# Drift-Free Orientation Estimation for Running

## Overview

This repository contains a Python implementation of the drift-free 3D orientation and displacement estimation method described in the paper:

**"Drift-Free 3D Orientation and Displacement Estimation for Quasi-Cyclical Movements Using One Inertial Measurement Unit: Application to Running"**

## Project Status

🚧 **Work in Progress** 🚧

This is an ongoing implementation attempt.


## Reference

Zandbergen, M. A., Reenalda, J., van Middelaar, R. P., Ferla, R. I., Buurke, J. H., & Veltink, P. H. (2022). Drift-free 3D orientation and displacement estimation for quasi-cyclical movements using one inertial measurement unit: Application to running. Sensors, 22(3), 956.
